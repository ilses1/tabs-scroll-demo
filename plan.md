# 前端React项目创建计划

## 项目概述
创建一个使用Vite、React 18、Umi、Ant Design Pro的后台项目模板，实现一个tabs滚动demo，包含以下功能：
- 顶部横幅显示时间和标题
- 下方有7个tabs
- 点击tab滚动到对应区域
- 每个tab区域高500px
- 滚动时横幅和tabs吸顶固定

## 计划步骤

### 1. 项目初始化
- 使用Vite创建React项目
- 配置React 18版本
- 安装Umi和Ant Design Pro依赖

### 2. 项目配置
- 配置Umi项目结构
- 配置Ant Design Pro主题
- 配置路由

### 3. 组件开发
- 创建顶部横幅组件（Header）
- 创建Tabs组件
- 创建Tab内容区域组件
- 实现滚动监听和吸顶效果
- 实现点击tab滚动到对应区域功能

### 4. 样式设计
- 设计顶部横幅样式
- 设计Tabs样式
- 设计Tab内容区域样式
- 实现响应式设计

### 5. 功能实现
- 实时时间显示
- Tabs切换逻辑
- 滚动监听和吸顶效果
- 平滑滚动功能

### 6. 测试和优化
- 测试功能完整性
- 优化性能
- 修复可能的bug

## 技术栈
- Vite
- React 18
- Umi
- Ant Design Pro
- TypeScript
- Less/CSS

## 预期效果
- 页面顶部显示带有实时时间的横幅
- 横幅下方是7个可切换的tabs
- 点击任意tab，页面平滑滚动到对应内容区域
- 向下滚动时，横幅和tabs固定在顶部
- 向上滚动时，恢复原始布局
